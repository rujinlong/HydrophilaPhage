## 2. 818 / 新兽药政策对接(⛔ 严守 EL-003 / DEC-D009)

**红线(勿犯)**:水产噬菌体走**农业农村部新兽药注册路径**,**不适用国务院 818 号令**(818 是**人用**生物医学新技术临床研究备案)。**绝不可写"水产噬菌体对接 818"**——这是 2025 评审级硬伤的同类误读。

**program / 申请书叙事的正确政策对接**:
- **线② 安全筛查 DB**(毒力 VFDB / ARG CARD / 溶原模块筛查)→ 直接产出**新兽药注册所需的安全性评估数据**(治疗噬菌体注册必需项)→ 这是 program 与"新兽药注册路径"的**实质对接点**,比泛泛提政策更硬。
- **资源平台**(线② + 综述)→ 对接 **NMDC(国家微生物科学数据中心)/ 数据要素×三年行动计划**。
- **宏观政策武器**:农业农村部 **194 号公告(2020 饲料禁抗)** + 健康中国 2030 + 十五五生物制造规划 + 水产绿色健康养殖"五大行动"。
- **818 只在人用语境提**:人用 BCoAT 噬菌体 PCT(WO 2025/104216 A1)可对接 818 人用备案——与水产线**分开陈述**,不混。

**一句话叙事(可进 2027 申请书)**:"建立面向**新兽药注册**的水产病原噬菌体**安全性评估资源平台**,产出毒力/耐药/溶原筛查数据,对接 NMDC 数据基础设施与农业农村部'禁抗替抗'国策。"(全程不碰 818。)
